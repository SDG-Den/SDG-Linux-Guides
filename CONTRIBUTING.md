Any human being is welcome to contribute to this project through the use of pull requests. 

# Guidelines

keep files in the right directories:
guides meant for new users go in newbie-guides
guides meant for other users go in other guides
any example files and resources (like example script snippets and config files, links to external resources etc) go in examples under their own subdirectory
any pictures go in pictures as well as videos go in videos, make a subdirectory for multiple related files.
any full scripts go in scripts, make a subdirectory for related files.


the language of the project is english and all files should be written in plain english.

avoid "technical speak" when possible, whenever a relevant technical term is used, assume that unless you tell the reader what it means, the reader will not know.
