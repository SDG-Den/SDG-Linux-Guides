# SDG-Linux-Guides


This repository is intended as an open-source resource for new and experienced linux users alike.

This is my first forray into using github for actual collaboration, the goal is to have various written guides and resources on different topics within linux.


## file structure of the project:


newbie-guides:

this contains any written guide aimed at new users, this means users who *just* installed their first linux distribution or have not tried linux at all yet. content in here should be applicable to most users.  

other guides:
this is for linux users looking to learn a bit more, and can contain more complex subjects that don't neccesarily apply to *most* users.

pictures/videos: for accompanying media. 

scripts: for any scripts this repo may provide, these would generally be examples referenced in guides.

examples: for any non-script example files referenced in guides, for example, this could be a configuration file of some kind. 



currently, the project is heavily WIP as it has just been started.
